* Features :

 -> Text Analysis :
   + Number of words
   + Number of characters
   + Number of sentences
   + Average word length
   + Most frequently used word

 -> Automatic Summarization :
   + Creates a short summary by selecting the longest and most informative sentences in the text.

 -> Mobile Compatible Interface :
   + Works smoothly on all devices, has a responsive design.

   + JavaScript
